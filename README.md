NeonBytes Store

NeonBytes Store is an online platform designed to showcase and explore various games.
Users can view popular titles, read detailed information, and access download links directly.

The project has been developed using React, React Router, Tailwind CSS, DaisyUI, and Firebase Authentication.
It offers a clean and responsive interface that ensures smooth navigation and a consistent modern theme.

Live URL: Not yet deployed

Key Features:
The homepage contains a banner slider, popular games section, and a newsletter form.
Users can create accounts, log in, and log out through Firebase authentication.
Each game has its own detailed page showing full information and a download button.
The profile page allows users to view and update their personal information.
An Esports News page displays the latest gaming and tournament updates from JSON data.
The website layout is fully responsive for mobile, tablet, and desktop devices.
